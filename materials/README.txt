Place your course PDFs here so the app can load them.

1. Copy each "Unit N" folder from your computer into this directory.
2. Keep the same subfolders and filenames as in your JSON (Homework, Notes, Review, etc.).
3. Example paths that must exist for FRQs and Notes "Packet PDFs" links:
   materials/Unit 1/Homework/1-1 WK A Change At An Instant #1.pdf
   materials/Unit 2/Homework/2-1 WK A Ave and Instantaneous Rates of Change #1.pdf

Some binders use a folder literally named "Homework " (with a trailing space). Paths in the JSON match that name; keep the same spelling when you copy files.
